# 專案簡介
## 專案目的
實作一個吃角子老虎機

###實作規則如下
1.三個滾輪的排列順序如下

| Reel 1 | Reel 2 | Reel 3 |
|--------|--------|--------|
|A | Z | A |
|B | Y | B |
|C | X | C |
|X | A | X |
|Y | B | Y |
|Z | C | Z |

2.每個符號代表的權重，權重越大出現機率越高
| Symbol Name | Symbol Weight |
|-------------|--------------|
| A | 1 |
| B | 1 |
| C | 1 |
| X | 2 |
| Y | 3 |
| Z | 4 |

3.滾輪的倍率獎金Mapping圖
| Winning Combination | Bet Multiplier |
|-------------|----------------|
| A,A,A | 20  |
| B,B,B | 20  |
| C,C,C | 20  |
| A,B,C | 30  |
